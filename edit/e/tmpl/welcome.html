This is an example application to show how to use Appudo on the lowest level. It is just html placed in the data area common to all accounts. As a non mandatory component it is not feature complete and has known bugs. It is possible to create and manage runs with this interface but be warned it is not pure fun.

Feel free to use the examples or add some new runs in the tree on the left.

Use Ctrl-S to save edited text.
